# Workspace setup

Persist the search in `~/job-search/` so nothing is re-derived each session.

## Create the files on first use

```bash
mkdir -p ~/job-search/materials ~/job-search/archive
touch ~/job-search/memory.md ~/job-search/applications.md ~/job-search/companies.md
```

## `memory.md`, preferences (read this first each session)

```markdown
# Job Search Memory

- Industries:

- Must-haves:
- Dealbreakers:

- Years of experience:
- Current / last role:
- Key strengths:
- Writing tone: [formal / conversational / technical]

## Where I'm looking
- LinkedIn: [on / off]
- Other platforms:
- Target-company list:

*Updated: YYYY-MM-DD*
```
