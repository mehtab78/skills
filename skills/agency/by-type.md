# Specifics by agency type

The operating loop is the same across agencies, but the metrics, deliverables,
scope-creep patterns, and tools differ. Use this to tune the defaults.

## Marketing

- **Watch:** client MRR/revenue, campaign performance (ROAS, CPA, CTR), content
  engagement, lead generation.
- **Deliver:** content calendars, social posts, ad creative and copy, performance
  reports, strategy docs.
- **Creep to catch:** "can you just post this too?", unlimited ad revisions,
  channel expansion with no budget bump.
- **Tools:** Meta Business Suite, Google Ads/Analytics, social schedulers, email
  platforms.

## Development

- **Watch:** sprint velocity, bug count and resolution time, deploy frequency,
  technical debt.
- **Deliver:** working software, technical docs, API specs, deployment guides.
- **Creep to catch:** "just one more feature," "bug fixes" that are really new
  features, integrations that weren't in scope.
- **Tools:** GitHub/GitLab, a project tracker (Linear, Jira), CI/CD, error
  monitoring.

## Design

- **Watch:** revision rounds used, time to first concept, client satisfaction,
  portfolio-worthy output.
- **Deliver:** brand identity, UI/UX, marketing collateral, brand guidelines.
- **Creep to catch:** "try a totally different direction," stakeholders added
  mid-project, print added to a digital-only scope.
- **Tools:** Figma, asset management, feedback tools (Loom, markup).

## Consulting

- **Watch:** recommendations implemented, measurable client outcomes, repeat
  business, thought-leadership output.
- **Deliver:** diagnostic reports, strategy docs, implementation roadmaps,
  executive presentations.
- **Creep to catch:** "can you help us implement too?", extra stakeholder
  interviews, an expanding diagnostic.
- **Tools:** a knowledge base, data-analysis tools, presentation tools.

## Content

- **Watch:** content volume, SEO rankings, engagement, client retention.
- **Deliver:** blog posts and articles, social content, video scripts, SEO
  reports.
- **Creep to catch:** extra platforms, shrinking review cycles, "can you also
  handle posting?"
- **Tools:** SEO tools (Ahrefs, SEMrush), content calendars, publishing
  platforms.

## Automation

- **Watch:** hours saved for the client, automation reliability/uptime, ROI
  delivered, support tickets.
- **Deliver:** workflow builds, integration setups, documentation, training
  materials.
- **Creep to catch:** "can it also do X?", legacy-system integrations, unspoken
  ongoing-support expectations.
- **Tools:** automation platforms (Make, n8n, Zapier), API connections,
  monitoring.
