# Client communication

Be proactive, be structured, and keep a written trail for anything that matters.
Match each client's register, formal or casual, and draft everything for the
user to approve before it sends.

## Weekly status

A predictable weekly note prevents most "what's the status?" pings. Cover:
progress this week, what's in progress (with expected dates), what you're waiting
on *them* for (with a by-when), what's planned next week, and any heads-up on
issues plus the proposed fix.

## Common situations

- **Ad-hoc status ask** → check the board; summarize the current phase, recent
  progress, next steps, and anything you need from them.
- **Chaotic feedback** → parse it into a clean list, tag each item bug /
  enhancement / scope change, and send it back for confirmation.
- **Scope-change request** → acknowledge it, assess hours/timeline/cost, and
  offer options: add with budget, swap for something equal, or defer to phase 2.
- **Slow to respond** → gentle nudge around day 3, a firmer note explaining the
  timeline impact around day 7, escalate or pause around day 10.
- **Unhappy client** → acknowledge the frustration first (don't get defensive),
  clarify the specific issue, propose a concrete fix, and have the user review
  the reply before it goes.

## Drafts to adapt

- **Confirming feedback:** *"Thanks for the detail, confirming I've got it: 1)…
  2)… Anything I missed before we proceed?"*
- **Scope change:** *"Happy to look at [request]. Heads up, it'd be [added
  scope/time/cost], want a quick change order, or save it for phase 2?"*
- **Deadline risk:** *"Heads up: [issue] is putting the [date] deadline at risk.
  I'd recommend [fix], how would you like to proceed?"*
- **Invoice nudge:** *"Floating this up your inbox, invoice [#] from [date] is
  still open. Happy to resend if anything's unclear."*

## Around meetings

Before: review status, list discussion points, prep any visuals, and share an
agenda in advance. After: summarize decisions and action items, send a recap
within a day, and update the board.
